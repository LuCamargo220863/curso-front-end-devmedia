Instalar o Visual Studio Code

Para começar a escrever os seus códigos, você precisa de uma ferramenta conhecida como ?Editor de código?. Essa ferramenta também é chamada de IDE. Nesse caso, recomendamos a instalação do Visual Studio Code (VS Code).

Para fazer o download do VS Code basta acessar esse link.

Veja o processo de instalação do VS Code no flow a seguir:






#PraCegoVer - Transcrição do flow - Instalando o VS Code No canto superior direito do site encontra-se o botão de download. Ao clicar nele o download do arquivo é feito e é feito o direcionamento para a página de instruções do VS Code. Ao clicar no arquivo baixado, uma tela de seleção da linguagem aparece ? escolha a linguagem e clique em ok.
Nas próximas telas utilize a sequência de cliques: next, install e Finish. O programa é executado na máquina e aparece na página de introdução.




Para mais informações sobre a instalação do VS Code, acesse o link desse artigo.

Criar a estrutura do projeto

Após instalar o VS Code, é aconselhável começar a organização do projeto.

Para isso, recomendamos que você crie a pasta e o arquivo necessário para a estrutura do seu projeto

Essa etapa deve ser realizada antes de trabalhar na parte visual da página.

Veja mais sobre isso no flow a seguir:






#PraCegoVer - Transcrição do flow - Estrutura do projeto A criação de um site deve sempre começar pela sua estrutura. Ou seja, você deve primeiro criar a(s) pasta(s) e arquivo(s) que você irá utilizar no projeto. A estrutura desse projeto será composta por uma pasta e um arquivo HTML. A pasta serve para organizar o projeto e o arquivo serve para conter o código HTML da página. A pasta do projeto será utiliza para armazenar os arquivos usados no site, deixando-o mais organizado. Já o arquivo HTML é usado para conter os elementos da página. Dentro do arquivo HTML inserimos o conteúdo que será exibido na página. Neste passo o foco é criar a pasta e o arquivo HTML. Somente depois vamos nos preocupar com o que será exibido no site. Faremos nessa ordem porque sem a pasta do projeto e o arquivo HTML não é possível inserir conteúdo na página do site: 1º passo: criar pasta e arquivo HTML; 2º passo ? inserir o código no arquivo HTML. Faremos nessa ordem porque sem a pasta do projeto e do arquivo HTML não é possível inserir conteúdo na página do site.




Após criar a pasta do projeto e o arquivo HTML, se você quiser abrir o site no navegador, ele terá a aparência apresentada na Figura 1.
