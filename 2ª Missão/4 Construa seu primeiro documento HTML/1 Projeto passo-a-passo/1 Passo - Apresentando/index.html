#PraCegoVer - Transcrição do flow - Conhecendo o projeto Nosso projeto é um site que contém informações sobre um parque. Esse site tem apenas um arquivo que contém diversas tags básicas do HTML, como a tag para o título, imagem, subtítulos e parágrafo. Por se tratar da sua primeira missão criando um projeto, não vamos nos preocupar com a estética, ou seja, não vamos criar estilos CSS. Nós vamos focar em posicionar e exibir os elementos HTML.



Para criar esse site, usamos algumas tags básicas do HTML, como vemos no flow a seguir:






#PraCegoVer - Transcrição do flow - Elementos utilizados no site Nesse projeto utilizamos diversas tags básicas para criar o site. Uma tag foi utilizada para representar o título do documento HTML (o nome que aparece na aba do navegador). Um quadrado ao redor do título do conteúdo (Parque Estadual Ilha Bela) é usado para destacar. Além disso, uma imagem também foi usada no arquivo HTML.
Abaixo da imagem temos um subtítulo no conteúdo do arquivo. Também temos outros subtítulos no arquivo, só que esses com menor relevância. Abaixo de cada subtítulo temos alguns elementos que representam parágrafos. Por fim, temos um link que foi adicionado no final do documento HTML.
