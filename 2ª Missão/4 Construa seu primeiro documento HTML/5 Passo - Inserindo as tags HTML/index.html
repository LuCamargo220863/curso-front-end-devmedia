#PraCegoVer - Transcrição do flow - Elementos visuais da página No passo anterior você criou a pasta do projeto (parque-estadual) e o arquivo HTML (index.html), além de adicionar um código base no arquivo HTML.
Ao fazer isso, a página do site ficou sem nenhum conteúdo. Nesse passo, vamos adicionar o código HTML que vai exibir os elementos na tela. Após adicionar o código HTML na página, ela ficará com a aparência apresentada na Figura 2.
Vamos inserir os elementos visuais agora porque já temos a estrutura básica da página pronta para isso - tudo ficará dentro da tag body.
Sem essa estrutura básica e sem o arquivo HTML não seria possível inserir os elementos HTML.


Como fazer?

Antes de implementar os elementos na página, veja mais detalhes sobre a imagem que será utilizada no projeto:





#PraCegoVer - Transcrição do flow - Imagem do projeto Como vimos antes, o projeto tem uma imagem que será usada na página. Essa imagem se encontra disponível para download no passo "Arquivos do projeto". Faça o download da imagem e salve a na pasta do projeto, no mesmo lugar do arquivo index ponto HTML (parque-ilha-bela.jpg).


Veja no flow a seguir como os elementos da página foram divididos em seções:





#PraCegoVer - Transcrição do flow - Divisão da página em seções A página do parque está dividida em 4 pequenas seções, oferecendo assim uma estrutura mais simples e organizada.
A primeira seção consiste no título da página (Parque Estadual Ilha Bela), seguida da imagem, um subtítulo e um parágrafo. A segunda, terceira e quarta seções consistem no subtítulo e parágrafo, cada.
Cada seção deve ser delimitada por uma tag que agrupa elementos. Assim, as seções ficarão separadas entre si.
Qual é a estrutura básica já criada, agora vamos adicionar os elementos visuais na página, ou seja, dentro da tag body.
As seções de todos os elementos da página (título, subtítulos, imagem e parágrafos) devem ser inseridas dentro da tag body.
No flow a seguir vamos ver como inserir os elementos nessas seções.







#PraCegoVer - Transcrição do flow - Inserindo elementos visuais na página Vimos que os elementos estão divididos em 4 seções e agora vamos inseri-los em cada seção. seguida da imagem, um subtítulo e um parágrafo.
A segunda, terceira e quarta seções consistem no subtítulo e parágrafo, cada.
Na primeira sessão temos um título que descreve o nome do local (Parque Estadual Ilha Bela).
Também tem uma imagem, que exibe uma paisagem do parque (lago com pedras grandes ao redor, rua com grandes arvores ao redor e uma pessoa parada na rua olhando pro lago).
Um subtítulo de maior relevância foi usado na primeira seção para descrever como é o local.
Temos um parágrafo no final da primeira seção que dá uma breve descrição do parque. Nas seções 2, 3 e 4 temos uma estrutura muito parecida, contendo um subtítulo e um parágrafo.
Nessas seções temos subtítulos de menor relevância que apontam características do local. Também temos parágrafos nessas seções que descrevem o local com maiores detalhes.
Na quarta seção temos um link usado como um meio de contato. Esse elemento cria um link para um telefone. Quando clicado, faz uma chamada para um número.
