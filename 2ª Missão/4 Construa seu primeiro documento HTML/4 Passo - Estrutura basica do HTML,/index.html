O que será feito?

O objetivo deste passo é inserir dentro do arquivo HTML, o código base referente a estrutura básica do documento.


Ao concluir este passo, a página do site terá a aparência exibida na Figura 1, apresentando apenas o título da página.

Para criar a estrutura básica do documento, você precisa inserir o código base no arquivo HTML, como mostra o flow a seguir:





#PraCegoVer - Transcrição do flow - Inserindo código base no arquivo HTML Nesse passo vamos criar a estrutura básica do HTML ao adicionar o código base no arquivo HTML. Ao criar uma estrutura básica vamos conseguir adicionar o título da página, aquele texto que fica na aba do navegador.
Para adicionar o código, primeiro abra o arquivo HTML criado (index.html). Dentro do arquivo HTML você pode adicionar o código base disponível no Código 1.
Com isso, você consegue alterar o título da página, através da tag title. Dessa forma, teremos a estrutura básica do HTML pronta para receber o conteúdo da página. Esse conteúdo ficará dentro da tag body.


O código utilizado representa a estrutura básica do HTML

Veja no Código 1 a estrutura básica do arquivo HTML

<!DOCTYPE html>

<html>

<head>

<title>Parque Estadual Ilha Bela</title>

</head>

<body>
