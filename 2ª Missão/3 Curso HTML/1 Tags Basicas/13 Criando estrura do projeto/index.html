13. Criando a estrutura do projeto

Antes de começarmos a digitar nosso documento HTML crie uma pasta para o projeto e baixe as imagens dos produtos (Figura 1) que estão disponíveis no botão Baixar Código Fonte.


Para criar um novo projeto, é essencial que você utilize um editor de texto para escrever os seus códigos.

Existem diversos editores que podem ser utilizados para escrever códigos de programação. Você pode até mesmo utilizar o Notepad do Windows.

Contudo, recomendamos o uso do editor de códigos Visual Studio Code (também chamado de VS Code). Você pode baixar o VS Code gratuitamente nesse link.

.

Se você precisar de mais informações sobre como usar o VS Code, temos um tutorial completo nesse link.
