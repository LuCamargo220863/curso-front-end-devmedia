4. Meta tag charset


Para definir a descrição de uma página precisamos utilizar a tag <meta name=”description”> dentro do cabeçalho HTML e é isso que vamos aprender nessa aula.









No Código 1 temos o exemplo utilizado no flow anterior.





<!DOCTYPE html>

<html lang="pt-br">

<head>

;

<title>Curso de HTML</title>

<meta name="description" content="Este é o curso de HTML onde você vai aprender tudo do básico ao avançado">

</head>

<body>


</body>

</html>


Código 1. Curso HTML


Veja na Figura 1 o uso da meta tag description.





As meta tags podem ou não utilizar '/' para o seu fechamento. O uso mais comum é sem a '/':

<meta name="description" content="Descrição">

Porém também é aceito o uso de '/':

<meta name="description" content="Descrição">



Exemplo prático

A meta tag description é utilizada por buscadores na hora de exibir o resultado da busca.
Veja nesse exemplo da Figura 2 como o buscador exibe a description no seu resultado.






<!DOCTYPE html>

<html lang="pt-br">

</head>

</body>
