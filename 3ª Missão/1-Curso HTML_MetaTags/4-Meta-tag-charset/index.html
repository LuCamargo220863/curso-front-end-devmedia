4. Meta tag charset

Para definir a descrição de uma página precisamos utilizar a tag <meta charset="código do charset"> dentro do cabeçalho HTML e é isso que vamos aprender nessa aula.




Perceba que a tag charset corrigiu o problema de acentuação na página.

No Código 1 temos a codificação do flow anterior.



<!DOCTYPE html>

<html>

<head>

<meta charset="UTF-8">

<title>Página do curso de HTML</title>

</head>

<body>

<h1>Introdução</h1>

<p>Este é o curso de HTML</p>

</body>

</html>


Código 1. Página do Curso HTML

Na Figura 1 podemos ver como a tag charset é utilizada.






Em alguns casos essa codificação é implícita no arquivo, com isso os caracteres da página não vão quebrar mesmo que a meta charset não for utilizada. Porém é recomendado que você sempre utilize essa meta tag.


Exemplo prático

A meta tag charset é utilizada para definir o conjunto de caracteres de uma página, e por vezes resolve problemas como a exibição incorreta de caracteres.
Veja nesse exemplo da Figura 2 como o buscador exibe a charset corrige a exibição de caracteres acentuados.


No Código 2 você pode ver o código utilizado no flow anterior.





<!DOCTYPE html>

<html lang="pt-br">

<head>

<title>Certificado de conclusão</title>

</head>

<body>


<h1>Conclusão de curso</h1>

<p>Você completou o curso de HTML</p>

</body>

</html>
