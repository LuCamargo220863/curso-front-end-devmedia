5. Meta tag Robots

Para definir a descrição de uma página precisamos utilizar a tag <meta name="robots"> dentro do cabeçalho HTML e é isso que vamos aprender nessa aula.




No Código 1 temos o exemplo com o uso do noindex.



<!DOCTYPE html>

<html>

<head>

<title>Comentários em aberto - DevMedia</title>

<meta name="robots" content="noindex" />

</head>

<body>

<h1>Relatórios de comentários em aberto</h1>

<p>Aqui entra o conteúdo do relatório</p>

</body>

</html>


Código 1. Comentários

Nofollow

Além do noindex, existe também um outro tipo de uso para a tag robots, o nofollow.







<!DOCTYPE html>

<html>

<head>

<title>Área de comentários</title>

<meta name="robots" content="nofollow" />

</head>

<body>

<h1>Área de comentários</h1>

<p>Aqui são exibidos todos os comentários</p>

</body>

</html>

Exemplo prático

Quando temos uma página administrativa, como um relatório de comentários, ou relatório financeiro, não queremos que essas páginas apareçam em resultados de busca, e por isso utilizamos a meta tag robots.

Veja na Figura 2 um exemplo de página administrativa que utiliza noindex e nofollow.


No Código 2 você pode ver o código utilizado no flow anterior.





<!DOCTYPE html>

<html>

<head>

<meta name="description" content="Relatório financeiro" />

<meta charset="UTF-8">

<meta name="robots" content="noindex, nofollow"/>

<title>Relatório financeiro</title>

</head>

<body>

<h1>Relatório financeiro</h1>

<div>

<p>Dados financeiros do segundo trimestre fiscal de 2021</p>

<p> As planilhas podem ser acessadas pelo <a href="https://www.office.com">office online</a> </p>

<p> Acesse outros dados do relatório no <a href="https://www.meusite.com/relatorimportante/">nosso sistema interno</a> </p>

</div>

</body>

</html>

Código 3. Uso do nofollow



NOTA: Utilizamos as meta tags description e noindex juntas neste exemplo como forma de mostrar o uso de todas as meta tags aprendidas até o momento, entretanto, ao utilizar noindex, o uso da meta tag description é opcional.
A meta tag robots é importante para indicar aos buscadores como queremos que os dados da nossa página sejam tratados



A tag meta charset é muito útil para definir o conjunto de caracteres da página, evitando problemas de acentuação
