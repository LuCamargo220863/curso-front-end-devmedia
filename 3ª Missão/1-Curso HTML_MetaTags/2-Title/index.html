Para definir o título de uma página precisamos definir a tag <title> dentro do cabeçalho HTML e é isso que vamos aprender nessa aula.











Como vimos, o uso da tag title exibe o título Minha Página definido no cabeçalho no navegador.
No Código 1 você pode conferir a codificação utilizada no flow anterior.



<!DOCTYPE html>

<html lang="pt-br">

<head>

;

<title>Minha Página</title>

</head>

<body>

</body>

</html>

Veja na Figura 1 a tag title exibida no navegador.





Caso os caracteres com acento na sua página não estejam aparecendo não se preocupe, na aula meta tag charset você vai aprender como resolver isso.


Exemplo prático

A tag title é muito útil para identificar onde o usuário está na página. Veja nesse exemplo da Figura 2 como a página identifica que o usuário está em um Curso de HTML






<!DOCTYPE html>

<html lang="pt-br">

<head>

;

<title>Curso de HTML</title>

</head>

<body>

<h1>Introdução</h1>

<div>

<p>

Bem-vindo ao curso de HTML, aqui você vai aprender tudo sobre HTML, do básico ao avançado.

</p>

</div>

</body>

</html>
