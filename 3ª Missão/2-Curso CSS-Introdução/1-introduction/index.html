1. Introdução ao CSS

Para mudar a aparência de uma página web é necessário utilizar uma linguagem de estilo chamada CSS (Cascading Style Sheets - Folha de Estilo em Cascata). Com ela podemos mudar o visual dos elementos que compõem um documento HTML.

No flow a seguir vemos o conceito de CSS:





Nos Código 1 e 2 vemos a codificação utilizada no flow.


Código 1. Código HTML



<h1>Mercado de Tecnologia aquecido</h1>

<a href="#">Contato</a>



Código 2. Código CSS



h1 {

font-size: 32px;

font-family: Verdana;

text-align: center;

}


a {

font-size: 14px;

font-family: Verdana;

text-align: none;

}





É fundamental para um desenvolvedor web ter conhecimentos de CSS.
