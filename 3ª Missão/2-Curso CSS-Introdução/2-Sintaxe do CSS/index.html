Cada estilo aplicado no CSS altera uma característica do elemento. Na Figura 1 podemos ver a propriedade color em destaque, que altera a cor da fonte do elemento.





Um elemento pode ter diversas propriedades configuradas. Como vimos no exemplo utilizado no flow, as propriedades font-size e color foram configuradas no elemento título. Isso fez com que o tamanho e a cor da fonte fossem alterados

É fundamental utilizar a sintaxe do CSS de forma correta para que os estilos sejam aplicados na página. Todos os estilos devem estar dentro de uma chave de abertura e uma chave de fechamento. Além disso, cada estilo é representado por uma propriedade e seu valor. Essa regra vale para qualquer elemento que terá a aparência mudada no código CSS
