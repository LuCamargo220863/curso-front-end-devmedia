4. Padrões de cor

Padrões de cores é um assunto muito relevante para um programador front-end. Isso porque as cores são constantemente usadas em todo tipo de projeto. Sendo assim, é um conhecimento fundamental para sua carreira como desenvolvedor front-end.

É muito importante que você conheça os principais padrões de cor que são usados no CSS. Basicamente utilizamos três deles como podemos ver na Figura 1.





Conheça no flow abaixo o padrão de cor nominal:





Na Animação 1 temos a listagem de cores disponíveis no padrão de cor nominal. No editor Visual Studio Code, a lista de cores pode ser acessada com as teclas CTRL + Espaço.





O padrão nominal utiliza o nome das cores em inglês

Agora vamos conhecer outro padrão de cores.
No flow a seguir vamos ver o padrão RGB:






O padrão RGB (red, green, blue) utiliza valores de 0 a 255 para indicar o quanto de cada cor se quer ao estilizar um elemento
